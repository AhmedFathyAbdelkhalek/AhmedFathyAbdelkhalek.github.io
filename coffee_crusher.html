Pneumatic Coffee Bean Crusher

This is a pneumatic coffee crusher that uses pneumatic cylinders to load, crush, and dispense coffee beans using classic control.

Origin

This project was part of my robotics course at the GUC. We were instructed to research service robots that help the elderly and then modify a Poppy humanoid arm to match the selected service robot application. This project was selected and done in collaboration with three teammates. Our application aims to help the elderly as well as anyone with tremor do everyday tasks in a reliable way.
